# B13 — Canned Responses Chatwoot: 52 Templates Jurídicos PT-BR

> Cartório 2º Notas Uberlândia — Agent AI Chatbot
> LGPD: NÃO incluir PII nos templates (CPF, RG, etc.)
> Formato: Chatwoot Canned Response importável

## Como importar

1. Acessar Chatwoot → Settings → Canned Responses
2. Criar cada response com shortcode + conteúdo
3. Shortcodes são usados pelo agent AI para respostas rápidas

---

## 📋 Acessos e Protocolos

### /novo-protocolo
```
Para criar um novo protocolo, preciso das seguintes informações:
1. Tipo de atendimento (emolumento, apostila, procuração, etc.)
2. Dados do cliente (nome completo)
3. Documentos necessários

Posso iniciar agora?
```

### /consultar-protocolo
```
Para consultar seu protocolo, informe o número do protocolo.
Exemplo: PROTO-2026-001234

Ou posso buscar pelo seu nome completo?
```

### /status-protocolo
```
Seu protocolo está {status}.

Próximos passos:
{proximos_passos}

Dúvidas? Estou à disposição.
```

### /documentos-necessarios
```
Documentos necessários para este serviço:
{lista_documentos}

Todos devem ser originais ou cópias autenticadas.
Traga também um documento de identidade com foto.
```

## 🏛️ Serviços Notariais

### /procuracao
```
Para uma procuração, preciso de:
1. Nome completo do outorgante
2. Nome completo do outorgado
3. Objeto da procuração
4. Poderes específicos (geral/especial)

Documentos: RG + CPF de ambos, comprovante de residência.
```

### /certidao-nascimento
```
Para emitir certidão de nascimento:
1. Nome completo
2. Data de nascimento
3. Nome dos pais
4. Naturalidade (cidade/UF)

Taxa: R$ {valor}
Prazo: {prazo}
```

### /certidao-casamento
```
Para certidão de casamento:
1. Nome completo (cônjuge 1)
2. Nome completo (cônjuge 2)
3. Data do casamento
4. Livro e folha (se disponível)

Taxa: R$ {valor}
```

### /averbacao
```
Para averbação, preciso de:
1. Documento a ser averbado
2. Dados para averbação
3. Valor da operação

A averbação é registrada à margem do documento original.
```

### /registro-imovel
```
Para registro de imóvel:
1. Matrícula atual (se existir)
2. Escritura ou documento de transmissão
3. Certidões negativas (IPTU, federais, cíveis)
4. Hartax pago

Documentação completa? Posso verificar.
```
