## J9 - Sentry SDK + PII scrubber

### Status atual
- backend/app/services/pii.py ja tem PII sanitizer (D8)
- Falta: integracao Sentry

### Plano
```python
import sentry_sdk
from app.services.pii import scrub_text_pii

def before_send(event, hint):
    if 'logentry' in event:
        event['logentry']['formatted'] = scrub_text_pii(event['logentry'].get('formatted', ''))
    if 'exception' in event:
        for exc in event['exception'].get('values', []):
            exc['value'] = scrub_text_pii(exc.get('value', ''))
    return event

sentry_sdk.init(
    dsn=settings.sentry_dsn,
    before_send=before_send,
    environment=settings.app_env,
    release=f"cartorio-api@{settings.git_sha}",
)
```
