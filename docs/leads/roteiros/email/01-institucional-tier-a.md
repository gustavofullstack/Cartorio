## Por que essa estrutura cumpre LGPD

| Requisito | Atendimento |
|-----------|-------------|
| **Consentimento específico** (art. 7º I, 8º) | Só continuamos após resposta afirmativa |
| **Informação clara** (art. 9º) | Identificação + finalidade + links da política/RIPD |
| **Finalidade explícita** (art. 9º II) | Demo + piloto 30d — declarado |
| **Opt-out facilitado** (art. 18 IX) | Palavra-chave "SAIR" sem custo |
| **Dado público** | Nome/cartório/e-mail vindos de site oficial |
| **Sem dado sensível** | Não pedimos CPF, RG, valor, endereço |
