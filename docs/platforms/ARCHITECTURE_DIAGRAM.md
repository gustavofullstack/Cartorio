## 3. Camadas da Arquitetura

### 3.1 Camada de Entrada (Edge)
- **Cloudflare DNS** вҶ’ Traefik SSL
- **Easypanel** вҶ’ Docker Swarm orchestration
- **Tailscale VPN** вҶ’ SSH seguro
- **LetвҖҷs Encrypt** вҶ’ Certificados auto-renovados

### 3.2 Camada de Gateway
- **Evolution API v2.3.7** (8080) вҶ’ WhatsApp Business API
- **Traefik** (80/443) вҶ’ Reverse proxy + SSL termination
- **5 eventos webhook** configurados

### 3.3 Camada de OrquestraГ§ГЈo
- **N8N v1.94** (5678) вҶ’ 34 workflows
- **N8N Runner** вҶ’ ExecuГ§ГЈo distribuГӯda
- **5 plugins** oficiais
- **5 provedores LLM** em paralelo (ZCODE.APP)

### 3.4 Camada de AplicaГ§ГЈo
- **API FastAPI v0.6.0** (8000) вҶ’ 58 endpoints
- **6 MCP servers** (164 tools)
- **JWT auth** + API Key (X-API-Key)
- **Pydantic V2** schemas
- **SQLAlchemy** ORM + **Alembic** migrations

### 3.5 Camada de Dados
- **Supabase** (5432) вҶ’ 134 tabelas (13 core + 121 suporte)
- **Redis** (6379) вҶ’ Cache + sessГөes + rate limiting
- **RLS ativo** em 4 tabelas crГӯticas
- **Outbox pattern** para events (pg_notify)
- **Triggers** para audit_log auto

### 3.6 Camada de Agent AI
- **OpenClaw Gateway** (18789) вҶ’ Pietra Agent
- **minimax-m3** (1M context, thinking adaptive)
- **7 skills** ativas
- **Provider opencode_go** (compat OpenAI, 1M tokens)
- **Thinking mode** adaptive

### 3.7 Camada de CRM
- **Chatwoot** (3000) вҶ’ WhatsApp + Email + Webchat
- **Sidekiq** вҶ’ Background jobs
- **HITL** (Human In The Loop) вҶ’ Pausar Agent
- **Macros/Canned Responses** вҶ’ AutomaГ§ГЈo
